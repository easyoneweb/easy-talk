# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [Unreleased]

### Added

- Project scaffold with Expo SDK 55 and TypeScript
- Authentication via Nextcloud Login Flow v2
- Conversations list with search, pull-to-refresh, and unread badges
- Chat window with message history, sending, replies, and real-time updates via long polling
- Contacts search with Nextcloud user autocomplete
- Settings screen with theme switching, notifications toggle, and logout
- Platform-adaptive UI: Liquid Glass (iOS 26+), BlurView fallback (iOS), Material Design 3 with dynamic colors (Android)
- Offline message and conversation caching with SQLite
- Push notification support scaffolding
- ESLint (eslint-config-expo) and Prettier integration
- Local build configuration (no EAS dependency)
- Deployment guide (`docs/DEPLOYMENT.md`)
